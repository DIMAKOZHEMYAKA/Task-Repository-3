# Task-Repository-3

Наблюдатель
----------------
**Описание**:  Это приложение позволяет запускать таймер, который через оповещения перезапускает анимацию каждые 20 секунд, и запускает видео спустя 2 секунды.
 - **Технологический стек**: JavaFX.
 - **Статус**:  1.0.
